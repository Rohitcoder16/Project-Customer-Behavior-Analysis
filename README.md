🛍️ Customer Shopping Behavior Analysis
An end-to-end data analysis project uncovering customer shopping patterns and business insights using Python, MySQL, and Power BI.

🔧 Tech Stack

🐍 Python (pandas, Seaborn, Matplotlib)
🗄️ MySQL
📓 Jupyter Notebook
📊 Power BI


📊 Dataset

3,900 rows | 18 columns
Covers customer demographics, purchase details, and shopping behavior


🔍 Project Pipeline

Python (EDA) – Data cleaning, missing value handling, feature engineering
MySQL (SQL) – 10 business queries (revenue by gender, customer segmentation, top products, etc.)
Power BI – Interactive dashboard with KPIs, revenue breakdowns, and slicers


💡 Key Insights

Male customers generate higher revenue than female customers
73% of customers are non-subscribers — big growth opportunity
Young Adults are the top revenue-generating age group
80% of customers fall in the Loyal segment


📈 Business Recommendations

Boost subscriptions with exclusive benefits
Reward loyal customers to improve retention
Focus marketing on high-revenue age groups
